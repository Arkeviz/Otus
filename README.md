# Проект создан в рамках курса OTUS JavaScript QA Engineer

## Установка
```shell
git clone https://github.com/Arkeviz/Otus.git
```
```shell
npm ci
```
## Команды
Запуск тестов
```shell
npm run test:run
```
Запуск `main.js`
```shell
npm run start
```
Линтер
```shell
npm run lint:fix
```

## Используемые технологии
- [Vite](https://vite.dev/)
- [Vitest](https://vitest.dev/) + [@vitest/ui](https://vitest.dev/guide/ui.html)
- [ofetch](https://github.com/unjs/ofetch)
- [faker-js](https://fakerjs.dev/)
- ESLint (конфиг [Anthony Fu](https://github.com/antfu/eslint-config))
